Simulador Comercial de Bosque

Este simulador a través de la consola permite seleccionar productos, definir cantidades, ver el subtotal y total a pagar con posibles descuentos cuando haya 5 o mas articulos del mismo.

La idea es poder integrar esta logica a la pagina web de un vivero (tienda de plantas) que desarrollaré para las siguiente entrega cuando aprendamos como interactuar con los elementos visuales del html y css
